# Global Superstore – Data Cleaning & Preparation

Data cleaning and preparation performed on the Global Superstore dataset as Task 2 of my Data Analytics Internship at **SkillCraft Technology**. The project focuses on transforming raw sales data into a clean, analysis-ready format using Python and Pandas.

##  Tools Used
- Python
- Pandas
- Google Colab

##  What Was Done
- Loaded the raw dataset (9,994+ records) using Pandas
- Checked for and validated missing values
- Checked for and validated duplicate rows
- Converted `Order Date` and `Ship Date` to proper datetime format
- Converted `Postal Code` to text (to preserve it as an identifier, not a numeric value)
- Exported the cleaned dataset as a new CSV file

##  Files
- `Sample - Superstore.csv` — original raw dataset
- `cleaned_superstore.csv` — final cleaned dataset
- `Untitled5.ipynb` — Colab notebook with all cleaning steps

##  Key Takeaway
Clean, correctly-typed data is the foundation for any reliable analysis — this task reinforced how much a proper cleaning step affects everything that comes after it.
